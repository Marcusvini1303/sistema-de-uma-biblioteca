# 📚 Sistema de Biblioteca em Python

Este projeto é um sistema de gerenciamento de biblioteca desenvolvido em Python utilizando Programação Orientada a Objetos.

## Funcionalidades

- Cadastro de clientes
- Cadastro de livros
- Remoção de livros
- Busca de livros
- Listagem de livros disponíveis
- Listagem de livros emprestados
- Empréstimo de livros
- Devolução de livros

## Tecnologias utilizadas

- Python
- Programação Orientada a Objetos (POO)
